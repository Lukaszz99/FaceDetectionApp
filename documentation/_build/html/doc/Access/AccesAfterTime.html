AccesAfterTime module¶

class AccesAfterTime.AccessAfterTime(min_time=2.5)[source]¶

Bases: object

This class provide access while person is recognized for X seconds
If access is granted, it is log into log file
If access is not granted or person is Unknown, screenshot is taken and this situation is also logged
Parameters

min_time (float) – minimum amount of time a person must be recognized to gain access.

access_after_time()[source]¶

Calculate person ratio in last min_time period. Ratio = score from get_persons_from_frame / frames read in min_time period.

Returns

with access and without access.

Return type

2 list of persons

access_in_location()[source]¶
Returns

True if access is grant in given location

check_time()[source]¶
Returns

True if min_time passed.

get_persons_from_frame(names, access, frame, video_stream)[source]¶

If person is recognized on frame add this name to dict, where all names are kept. If person has access add +1 in this name. If person don’t have access add -1 in this name. Names with negative value of person_ratio (access_after_time method) are recognized as persons without access.

Parameters

  • names (list) – list of names people on frame. Can contain ‘Unknown’ if person is not in known persons .pickle file.

  • access (list) – list of tuples (Name, Access(True/False)). Used to measure person_ratio.

  • frame (numpy.ndarray) – last frame with unknown or person with no access.

  • video_stream (WebCamStream) – source of video.

log_access(video_stream)[source]¶

If min_time passed, save into log.txt file who get access or not. Also save from video stream for person without access or unknown person. If someone get access location_access variable. is set to True. After 1sec is set to False.

Parameters

video_stream (WebCamStream) – source of video.
